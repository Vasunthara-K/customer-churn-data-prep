# Customer Churn Data Preparation

This is a hands-on learning project created to strengthen my Python and Pandas skills through a real-world data preparation workflow.

Instead of learning Python concepts in isolation, I am applying them to a Customer Churn dataset and performing activities similar to those carried out by Data Analysts and Data Scientists before model building.

## Learning Goals

* Practice Python using real datasets
* Learn Pandas through hands-on exercises
* Understand data quality assessment techniques
* Perform data cleaning and preprocessing
* Learn feature classification and business understanding of data
* Build a production-ready cleaned dataset

## Dataset

Telco Customer Churn Dataset

* Rows: 7043
* Columns: 21
* Target Column: Churn
